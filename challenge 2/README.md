<h1 align="center">SMS_AntiTheft</h1>

<p align="center">
  <img src="https://cloud.githubusercontent.com/assets/12548071/24226460/588e77aa-0f5e-11e7-8326-6e90c6ca4f8e.png" width="250" height="250">
</p>

Android Application perform these commands without internet (SMS):

* <b>`[ LOCK ]`</b>
* <b>`[ ERASE ]`</b>
* <b>`[ Wifi_ON ]`</b>
* <b>`[ Wifi_OFF ]`</b>
* <b>`[ Wifi_Details ]`</b>
* <b>`[ Location ]`</b>
* <b>`[ Silent_Mode ]`</b>
* <b>`[ Normal_Mode ]`</b>
* <b>`[ Bluetooth_ON ]`</b>
* <b>`[ Bluetooth_OFF ]`</b>
* <b>`[ Phone_Number ]`</b>
* <b>`[ Display_Lock ]`</b>
* <b>`[ SIM_Serial_Number ]`</b>

App is published on <b>`Google Play`</b>. You can download it for free from the following link:

```
URL_COMING_SOON
```

<h1>Inbuilt Client Manager</h1>

<p>Inbuilt client manager encrypts the command into a HMAC format and sends the command to the lost device and makes it unreadable</p>

<h1>Unencrypted Command (For Emergency Purpose)</h1>

<p>As an alternative way, user can also send a command without encrypting, in case of emergency. This method doesnâ€™t require the client manager tool or app to be installed on an alternative device.  </p>

<h3>Commands</h3><br>

```
smstheft password silent
smstheft password normal
smstheft password lockdisplay
smstheft password wifion
smstheft password wifioff
smstheft password wifiname
smstheft password bluetoothon
smstheft password bluetoothoff
smstheft password location
smstheft password siminfo
smstheft password phonenumber

```
